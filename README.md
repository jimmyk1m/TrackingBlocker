使用教程
- [点此](https://cdn.jsdelivr.net/gh/jimmyk1m/TrackingBlocker/main.mobileconfig)，确保弹窗 "已下载描述文件"
- 选择 "设置"-"已下载的描述文件"，并按屏幕指示输入密码安装
- 打开 "设置"-"通用"-"VPN与设备管理"
- 确保 "访问限制和代理"-"DNS"已选择为"Phone detection Blocklist"
- 恭喜您屏蔽已经生效
